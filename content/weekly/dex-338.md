## Design Skills 总结

### Impeccable

增强版 frontend-design skill，17 个设计命令（/polish、/audit、/distill 等）。

https://impeccable.style/

### UI Skills

包含 15 个独立 skills：baseline-ui（Tailwind 设计一致性）、fixing-accessibility（无障碍审计）、fixing-metadata（SEO 元数据）、fixing-motion-performance（动画性能）、12-principles-of-animation（迪士尼动画原则）等。

https://www.ui-skills.com/

### Taste Skill

包含 taste-skill（前端设计）、redesign-skill（升级现有项目）、output-skill（强制 AI 写完整代码）、soft-skill（让东西看起来和感觉昂贵。涵盖高级字体、宽阔的空白、层叠的卡片设计、流畅的弹簧动画以及漂浮导航。）。

https://github.com/Leonxlnx/taste-skill

### Better Icons

MCP server + skill，搜索并获取 20 万+ 图标，支持自动学习常用图标库，直接同步到项目 icons 文件。

https://github.com/better-auth/better-icons

### UI Design Brain

60+ 组件最佳实践库，包含布局模式、无障碍规则和反模式，支持 5 种设计风格（SaaS、极简、企业、创意、数据仪表盘）。

https://github.com/carmahhawwari/ui-design-brain

### AI kit by Motion (付费)

Motion+ 会员专属，包含动画专家 skill、性能审计、CSS spring 生成、过渡可视化和文档搜索工具。

https://motion.dev/docs/ai-kit

### UI UX Pro Max

包含 50+ 设计风格、97 种配色方案和 9 种技术栈，提供全面的 UI/UX 设计智能。

https://github.com/nextlevelbuilder/ui-ux-pro-max-skill

### Anthropic frontend-design

Anthropic 官方 skills 仓库，包含创意设计、技术开发、企业沟通等多类示例 skills，以及文档处理 skills（docx、pdf、pptx、xlsx）。

https://github.com/anthropics/skills

### Vercel web design guidelines

Vercel 官方 skills 集合，其中 web-design-guidelines 包含 100+ UI 审计规则。

https://github.com/vercel-labs/agent-skills

### Designer Skills Collection

63 个 skills 和 27 个命令，涵盖 8 个插件：用户研究、设计系统、UX 策略、UI 设计、交互设计、原型测试、设计运营和设计工具包。

https://github.com/Owl-Listener/designer-skills

### Emil Skills ($99)

animation.dev 推出的动效 skills，需要单独付费。

https://animations.dev/learn/emil-skill

### Skills by InterfaceCraft ($199)

需要购买这个课程才能使用。

https://www.interfacecraft.dev/

## Resources

### macOSicons

很老的网站了，收录超过 30,000 个由设计师社区贡献的免费 macOS 风格图标，主要是上周新改版了，以防有人不知道。

https://macosicons.com/

### Grainrad

一个基于浏览器的创意媒体处理工具，支持将图片、视频、GIF 甚至 3D（GLB）文件实时转换为 ASCII、抖动、半调、轮廓检测、像素排序、VHS 等十五种以上风格化效果，内置预设系统，并可导出为 PNG、SVG、MP4 等多种格式。

https://grainrad.com/

### OpenUI

一个面向生成式 UI 的开源标准工具包，允许开发者用 defineComponent 注册自己的组件库，由 LLM 在对话中直接返回结构化 UI 而非纯文本。兼容 ShadCN、Material 等设计系统，支持 OpenAI、Anthropic、Gemini 等主流模型。没准哪天被 OpenAI 收购了。

https://www.openui.com/

### 其他资源

- **300 FUI Elements Pack** - 一个包含 300 多个未来感界面元素的素材包，面向 After Effects、剪辑、影视和动态设计场景。https://nawazalamgir.gumroad.com/l/300fui
- **DesEngs** - 一个面向 Design Engineer 的资源导航站，聚合组件库、案例和参考资料。内容还是太少了，比我自己总结的差远了，等我抽时间也上线一个。https://desengs.com/

## Cases

### Natural

面向 AI Agent 的支付基础设施，设计风格有点模仿 Cursor，但还是很精致。

https://www.natural.co/

### mem_brane

本周最佳作品，通过自指循环、环境语音吸收和透明 LED 结构，把 AI 心智可视化成一种可感知的空间体验。看到这样的作品，我还真不能 vibe 出来，一定要点开视频看看。

https://www.nonfigurativ.com/projects/mem_brane

### Locale

前不久那个设计教程 interfacecraft 的作者 vibe 了一个小工具，一个调用本地字体权限来展示本机字体族的网页实验，方向更偏浏览器能力与排版环境探索。设计师的品位就是不一样。

https://joshpuckett.me/locale

### Matthew Petersen

很喜欢这个个人网站设计，展示的几个项目也很有创意。

https://www.matthewpetersen.ca/
