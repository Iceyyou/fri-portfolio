本期封面来自 km5 日本音频产品品牌，主打便携 CD 播器和轻量级头戴耳机。

上周科技圈传来噩耗，Tailwind CSS 裁员 75%，原因是 AI 冲击导致其商业产品营收暴跌。乍一听很夸张，其实一共就 4 个人，走了 3 个。对于他们的故事我是比较了解的，当年 Adam 想做的是 Digest 一个类似 HackerNews 的东西，后来还直播写代码，可以说是最早实践 build it in public 的独立开发者之一了，人们对他的产品兴趣不大，但发现他用的这个 CSS 框架真牛逼，于是就开始了真正的创业之路。

期间还做了在业界人手必备的设计入门书籍「Refactoring UI」，当年这两个项目就让作者赚疯了，一年至少几百万美金吧。但这么多年 Tailwind CSS 的唯一收入手段只有 Tailwind UI，299 一套还是永久买断版，还有 team 版，我们当年就组织过 3 波团购了，而现在更新很慢几乎处于不发展的状态了，AI 爆发后谁还需要这种模板呢？

Tailwind CSS 这么响亮的品牌本应开发出更多商业模式的，Vue 他们的母公司就是很好的例子。但开源项目本来变现就困难，CSS 又不是核心语言，感觉创始人也有点不思进取......不过仅仅一天各大互联网巨头都纷纷赞助支援了，也许在 AI 时代，这种互联网基础设施最终的结局都是被大公司收购吧。

Tailwind CSS 的普及让人们彻底不写 CSS 了，以前人们写各种语言，现在人们只写 markdown 了。

上周 dub.co 创始人 Steven Tey 分享将动画从 Rive 重构为 Lottie 后文件减小 90%，每月节省 1000 美元以上的故事，所以说用什么工具，哪个工具好用，只是设计师和工程师一厢情愿的事情，真正到了生产环节，如果网站规模和流量足够大，哪个省钱用哪个。
