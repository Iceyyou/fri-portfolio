本期封面来自 Spotify，他们已经移除了苹果 glass 风格，不言而喻这次苹果是真的失败了。

Paper 正在内测 MCP 功能，未来还会有桌面客户端，我已经拿到了邀请，正在体验中，虽说可以通过 MCP 和 Claude Code 联动，但还要打开终端使用起来还是挺割裂的。据说未来也会推出 skills 等内容，未来已来，靠嘴做设计这件事真的不远了。

期待了好久的 variant.ai 终于正式上线了，看官方给的案例和网友制作的，设计水平确实很高，当然也可能是预制了一系列这样高质量的设计风格提示词。但除了设计本身，它还试图颠覆人们熟悉已久的 chat 对话交互逻辑，改用看似更丝滑的滚动刷新生成设计。我不知道别人什么感受，反正我使用起来觉得怪怪的，看到有人也在说，不太喜欢这种像是在浏览网页的方式做设计。而且这种操作模式非常费 token，一不小心就给你刷新自动生成了 8 个新设计，幸好现在可以免费用不然真烧不起啊。

OpenClaw 没什么好说的了，还没玩过的人可能是赖，还不知道的可能真的活在古代了。至于那个 moltbook 为什么那么创新也可以理解，因为作者之前是币圈的，还发过币，很多 web3 领域的开发者和产品创意都非常厉害，现在他们很多人都转投 AI 赛道了，不知道这是拯救了他们还是造福了传统 web2 行业，没有炒币没有空投，专心做有用的产品才会获得更多的关注。

## Resources

### HeroUI Figma Kit V3

HeroUI 官方发布的 Figma 设计套件第三版，包含完整的组件库和设计系统资源，适合快速搭建基于 HeroUI 的界面原型和设计稿。

https://www.figma.com/community/file/1546526812159103429/heroui-figma-kit-v3

### Supa Colors

Supa 插件作者的新作，基于 OKLCH 色彩空间的专业配色系统工具，支持 P3 广色域显示，内置 WCAG 无障碍检查功能，可一键导出 CSS、Tailwind v4、TypeScript、Figma Variables 等 10+ 格式的生产级代码。但是又是订阅制按年收费，有点贵。

https://supacolors.studio/

### Pattern Playground

交互式图案生成器，提供 Metaballs 背景和丰富的动画效果（涟漪、波浪、漩涡、螺旋、噪点、潮汐等），支持自定义网格参数、混合模式调整和导出功能，适合创建独特的背景纹理。

https://patterns-three-livid.vercel.app/

### dpark's origami archive

dpark 把他自己以前的所有在 iOS 端的概念探索和各种创意项目方面的动画作品都公开了，档案中的每个条目都包含一个视频预览、作品标题、创作年份以及一个指向 Google Drive 的下载链接。

https://origamiarchive.com/
