## Glaze

Raycast 上周发布的桌面应用主题定制工具 Glaze 还挺惊艳的，个人感觉比小龙虾更有价值，做大了没准比 App Store 更有前景，当然被苹果收购也不是不可能。未来人人都能靠嘴生产 app 并上架应用商店。目前大部分人还没有测试邀请，不过可以先下载他们的新 Glaze 系列壁纸。

https://www.glazeapp.com/

## Agents UI

LiveKit 发布了一套语音交互的 UI 框架 Agents UI，和 elevenlabs 家前不久开源的那个有点类似。看到官网的 shader 动画挺好看的，大概是 Unicorn Studio 的那个 Aura 效果加上自定义的代码，于是我让 Claude Code 分析了一下代码，很快就复刻了出来，然后根据自己的需求又扩展了不同主题颜色和不同纹理。

在以前这简直是不敢想的事情，作为设计师，即使以前有代码开发经验，看到网上那些复杂的动画效果也很难完美实现，更不要说还要花很多时间学习，如今，任何看到的效果都可以尝试复刻，大部分在几分钟就完成了。有了 AI，我感觉自己无所不能。

https://livekit.io/ui

## Resources

### ASC11

在线 ASCII 艺术生成器，支持将图片和视频转换为多种风格的 ASCII 艺术，包括经典字符、盲文、半色调、粒子等模式，内置 10 段均衡器式的参数调节、多种抖动算法、色彩模式和 FX 特效预设，还可导出为 Canvas 或 WebGL 格式。

https://asc11.com/

### ASCII Clouds

另一个交互式 ASCII 云朵生成器，通过噪声算法实时渲染 ASCII 字符组成的云层动画，支持调节单元格大小、波幅、速度、色相、对比度等参数，内置多种预设风格如终端、复古 CRT、宇宙和雾气等。

https://caidan.dev/portfolio/ascii_clouds/

### The Gridcn

受电影《创：战纪》启发的 shadcn/ui 主题组件库，提供 6 套以希腊神话命名的霓虹色彩方案、50+ 组件、HUD 界面元素和 Three.js 3D 特效，支持 React + Tailwind + TypeScript，完全免费开源。

https://thegridcn.com/
