本期封面来自苹果传奇设计师 Susan Kare 为 OpenClaw 设计的可爱形象。

上周 Figma 发布了矢量化图像编辑，可以直接在 Figma 画布上将光栅图像转换为完全可编辑的矢量。这下更没有使用 Adobe 的必要了：

- **Convert images in one click** 一键转换图像：将任何光栅图像（例如草图、照片或纹理）拖放到 Figma Design 或 Draw 中，并立即将其转换为可编辑矢量。
- **Simplify color output** 简化颜色输出：使用"重新着色"滑块将矢量化图稿简化为一组可管理的颜色和形状，从而更容易优化和调整。
- **Edit with existing Figma tools** 使用现有 Figma 工具进行编辑：调整锚点、重新定位各个元素，并应用颜色变量来标准化跨主题和品牌系统的图稿。
- **Create reusable assets** 创建可重复使用的资源：将手绘字体转换为可编辑徽标，或将捕获的纹理转换为可扩展的叠加层，以实现分层 UI 设计和登陆页面。

## Resources

### TSL Graph

Three.js 可视化着色器编辑工具，提供节点式图形界面，让开发者无需手写代码即可创建强大的 Three.js 着色器，并可一键导出标准 TSL 代码。

https://www.tsl-graph.xyz/

### Introducing Geist Pixel

Vercel 推出 Geist 字体家族新成员 Geist Pixel，一款基于像素网格构建的位图风格字体，包含 Square、Grid、Circle、Triangle、Line 五种变体，480 个字形，支持 32 种语言。每个字形均经手工逐像素打磨，兼具复古质感与现代产品级可用性。未来还会有 serif 字体推出。

https://vercel.com/blog/introducing-geist-pixel

### CMYK Halftone

Figma 半色调效果插件，可模拟传统 CMYK 半色调印刷工艺，分别生成并叠加青、品红、黄、黑四色半色调网点图层，支持调节网点大小和角度等参数。正好上周 Paper 也发布了类似的 halftone 效果。

https://www.figma.com/community/plugin/1584204863720680047/cmyk-halftone
